#### for문
  - ``` java  
  	    for( 초기화식; 조건식; 증감식) {  
			    // 조건이 참일 경우 내부 반복 실행  
		    }  
      ```  
      
#### while문
  - ``` java  
        while(조건식) {  
          // 조건이 참일 경우 내부 반복문 실행  
        }  
        ```  
        
#### do-while문
  - ``` java    
        do {  
          // 조건이 참일 경우 내부 반복문 실행  
        }while(조건식)    
        ```  
        
  - 여기서 while과 do-while의 차이점은 while문은 조건식이 맞지않으면 실행되지 않지만 do-while은 조건식의 영향을 받지않고  
  무조건 1번이상 실행이된다는 것.
