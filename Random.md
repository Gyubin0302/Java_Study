#### Random
  - Random class는 난수를 생성하는 class로 객체를 생성해서 사용  
  - import를 통해 외부 class를 호출. import java.util.Random;  
  - nextInt(), nextDouble, nextBoolean등의 method 들이 있다
  - 매개변수값을 넣어주지 않으면 최소 값 ~ 최대 값 사이의 값이 출력
